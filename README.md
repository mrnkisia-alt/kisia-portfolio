# Collins Kisia - Personal Portfolio

A modern, minimal, and highly captivating personal portfolio website built entirely from scratch without any heavy frameworks. This site is designed to highlight my technical skills, professional services, and provide quick ways for collaboration.

## 🚀 Built With
- **HTML5:** Semantic and accessible structure.
- **CSS3:** Advanced modern styling including CSS Variables, Grid, Flexbox, Glassmorphism (backdrop-filters), CSS Animations, and dark-theme gradients.
- **Vanilla JavaScript:** Smooth scrolling and `IntersectionObserver` for sleek scroll-reveal animations.
- **Phosphor Icons:** Minimalist, scalable vector icons.
